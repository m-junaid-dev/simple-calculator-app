 Simple Calculator App

A lightweight calculator application designed to perform basic mathematical operations with an easy-to-use interface. This project is ideal for beginners who want to understand application logic and user input handling.

Features

Addition

Subtraction

Multiplication

Division

Clear / Reset functionality

Simple and clean UI

Technologies Used

HTML
CSS
JAVASCRIPT



Project Purpose

This project was created for learning and practice purposes. It helps in understanding basic programming logic, conditional statements, and user interaction.
